# Astro Website

Production BePartnerLabs site built with Astro 6.

## Stack

- **Astro 6** — static site generator
- **Node** ≥ 22.12.0

## Commands

```sh
npm run dev      # dev server at localhost:4321
npm run build    # production build
npm run preview  # preview production build locally
```

## Design Reference

Designs come from `../reference/be-partner-labs/project/`. Before implementing any UI:

1. Read chat transcripts in `../reference/be-partner-labs/chats/` for design intent
2. Implement HTML prototypes pixel-perfectly — match visual output, not prototype structure
3. Use `../reference/be-partner-labs/project/tokens.css` for design tokens (colors, spacing, typography)
4. Assets are in `../reference/be-partner-labs/project/uploads/`
